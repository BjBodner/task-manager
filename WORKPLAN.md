> מטרה: אפליקציה עובדת מקצה לקצה — קנבן, CRUD, שמירה וטעינה מ-Supabase.

### 1.1 הקמת פרויקט
- [ ] אתחול React + Vite + Tailwind CSS
- [ ] הגדרת RTL גלובלי ופונט עברי
- [ ] חיבור Supabase — יצירת פרויקט, הגדרת client ב-`lib/supabase.js`
- [ ] יצירת טבלאות ב-Supabase: `tasks`, `user_stats` (לפי מודל הנתונים ב-CLAUDE.md)

### 1.2 לוח קנבן בסיסי
- [ ] קומפוננטת `Board` — 4 עמודות RTL עם כותרות וצבעים
- [ ] קומפוננטת `TaskCard` — כרטיס עם כותרת, קטגוריה (צבע), עדיפות (אייקון)
- [ ] גרירה בין עמודות עם `@hello-pangea/dnd`
- [ ] עדכון סטטוס ב-Supabase בזמן אמת אחרי גרירה

### 1.3 ניהול משימות (CRUD)
- [ ] קומפוננטת `AddTask` — טופס הוספה מהירה (כותרת + קטגוריה + עדיפות)
- [ ] עריכת משימה — לחיצה על כרטיס פותחת מצב עריכה (inline או מודאל)
- [ ] מחיקת משימה עם דיאלוג אישור
- [ ] שמירה מיידית ל-Supabase בכל פעולה

### 1.4 טעינה ופרסיסטנטיות
- [ ] טעינת כל המשימות מ-Supabase בעלייה
- [ ] מצב טעינה (loading state) בזמן fetch
- [ ] טיפול בשגיאות רשת (הודעת שגיאה למשתמש)
- [ ] וידוא: סגירת דפדפן ופתיחה מחדש — כל המשימות נשמרות

### 1.5 Header בסיסי
- [ ] שם האפליקציה (TaskFlow)
- [ ] מקומות שמורים ל-XP bar, רמה, סטריק (ריקים בשלב זה)

### ✅ קריטריוני סיום שלב 1
